## ТЗ: РИЛС 1 — CLAUDE SECURITY

**Длина:** 28 сек
**Формат:** Split screen (лицо слева / экран справа)

---

**Покадрово:**

| Тайм | Экран | Голос |
|------|-------|-------|
| 0–3с | Лицо фронт, крупный план, стоп-кадр на 0.5с потом зум-ин | *«ИИ теперь взламывает сам — и Anthropic признала это официально»* |
| 3–8с | Split: лицо слева / справа браузер — официальная страница Anthropic Claude Security | *«Они запустили Claude Security в бета»* |
| 8–14с | Split: лицо слева / справа — экран с таймером, цифра «часы → секунды» анимацией | *«Раньше хакер тратил часы. Теперь ИИ делает это за секунды»* |
| 14–22с | Split: лицо слева / справа — интерфейс Claude Security, курсор движется | *«Anthropic закрыла Mythos и выкатила Claude Security для корпораций — чтобы ИИ защищал, а не ломал»* |
| 22–28с | Лицо фронт, зум-ин до глаз | *«Сохрани — пригодится»* |

---

**Субтитры:**
- Шрифт: **Bold sans-serif** (Montserrat ExtraBold или аналог)
- Размер: крупный, 1–3 слова в строке
- Цвет: белый с чёрной обводкой 2px
- Позиция: нижняя треть экрана
- Ключевые слова выделить: «СЕКУНДЫ», «MYTHOS», «CLAUDE SECURITY» — жёлтым или красным

---

**Музыка:**
- Трек: тревожный электронный фон, низкий бас
- Громкость: 15% под голос, на хуке (0–3с) громче — 35%, потом плавный fade down
- На финале (22–28с) музыка обрывается резко

---

**Зумы:**
- 0с — зум-ин +8% за 0.5с (хук)
- 14с — микрозум +5% на слове «закрыла»
- 22с — зум-ин до глаз, плавно за 1.5с

---

**Переходы:**
- 3с — hard cut на split screen
- 22с — hard cut обратно на лицо
- Никаких растворений, только резкие cut'ы
