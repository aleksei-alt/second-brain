## Выводы по формулам

### Что залетает (>10k просмотров)

**Формула #1 — личный воркфлоу + боль**
> "Делегировал / Перестал сливать / Стал получать результат в X раз быстрее"
- Конкретный инструмент + конкретная проблема которую он решает
- Пример: лимиты на PDF → Markdown workflow

**Формула #2 — демо результата + минимализм в хуке**
> "Claude Code позволяет создавать [результат] всего за одну команду"
- Wow-эффект: дорогой результат за минимальное действие
- 861 комментарий = высокое вовлечение, тема Claude Code resonates

**Формула #3 — список + Claude Code**
> "Отобрал 5 [инструментов], которые научат / помогут / ускорят"
- Число в хуке (5, 7, 10)
- Конкретный бенефит за каждым пунктом

**Формула #4 — curiosity gap**
> "Скрытые возможности / То, о чём никто не говорит"

### Что НЕ работает (<1k просмотров)
- Новостной формат без демо (#7, #8) — 684-788 просмотров
- Обобщённые темы без конкретного воркфлоу
- Нет чёткого хука (пустой caption #9)
