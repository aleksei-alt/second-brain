## 🛠 Топ-3 инструмента / фичи для внедрения

### 1. Claude Code + Composio
**Что:** Подключить Claude Code к внешним сервисам (Notion, Slack, GitHub, и т.д.) через CLI без написания интеграций вручную.
**Зачем:** Автоматизируешь рутину за минуты вместо дней.
**Как начать:** `npm install -g composio-core` → `composio add notion` → дать задачу Claude Code.

### 2. Claude в Adobe / Blender
**Что:** Нативная интеграция Claude в творческие инструменты.
**Зачем:** Генерация текста, описаний, prompt'ов прямо в рабочем пространстве — не переключаешься между окнами.
**Как начать:** Обновить Adobe Creative Cloud → найти Claude-коннектор в плагинах.

### 3. Claude Code → Pinterest Scrape → Moodboard
**Что:** Скрипт который берёт референс, идёт в Pinterest, скачивает 30-50 похожих изображений, собирает папку.
**Зачем:** 15 минут вместо 2 часов ручного поиска референсов.
**Как начать:** Дать Claude Code задачу с референс-картинкой + промпт для Pinterest scraping через Playwright/Puppeteer.
