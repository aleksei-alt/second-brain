## ЧТО МЫ ПРОДАЁМ

**Продукт:** AI-ассистент для малого бизнеса — отвечает клиентам 24/7 в WhatsApp/Telegram/Instagram, записывает, напоминает, квалифицирует лидов.

**Не "AI-бот".** Клиент покупает: не потерять клиента пока занят. Не технологию — результат.

**Позиционирование:** "Сотрудник который отвечает в 22:00 и в выходные без зарплаты."

---

## КОМУ МЫ ПРОДАЁМ

### Приоритет по нишам (от горячего к холодному):

**1. Beauty / Nail (горячая ниша)**
- 10-20% записей = no-show → **$67,000/год потерь** на средний салон
- 46% записей приходят вне рабочих часов
- Владелец — один человек, сам мастер, некому отвечать
- Принимают решения быстро, проблему понимают сразу
- Прайс: €100-150/мес + €150-300 setup

**2. Стоматология (горячая ниша, высокий чек)**
- 1 потерянный пациент = €850-1,500 (консультация + лечение)
- 28% малых клиник — не заполнены из-за no-show
- 62% звонков остаются без ответа
- Маленькие практики (1-2 врача) без администратора ночью
- Прайс: €150-200/мес + €200-400 setup

**3. Риелторы (горячая, высокий ROI)**
- 1 пропущенный лид = €500-10,000 комиссии
- Лиды падают в WA/TG → теряются в переписке
- На показе = не отвечает = клиент ушёл к другому агенту
- Прайс: €200/мес + €300-400 setup

**4. Юристы (тёплая)**
- 45% времени — non-billable admin и первичные вопросы
- <3 оплачиваемых часов из 8 рабочих
- Прайс: €150-175/мес + €200-350 setup

**5. Рестораны/Кафе (холоднее)**
- No-show 5-20% резерваций
- Google отзывы: 88% выбирают тех кто отвечает
- Прайс: €120/мес + €200 setup

**Убрано навсегда:**
- Цифра "68% пишут вне рабочих часов" — придумана, заменена на 46% и 62%
- Абстрактный оффер "я делаю ботов" — заменено на конкретные деньги

- Bulk Hiker API запросы — только по конкретному запросу оператора
- n8n как продукт — не продаётся
- Instagram DM для B2B аутрича — мёртв (145 DM → 2 ответа)
